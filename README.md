# biwenger-utils

## Tecnologías y Herramientas
- **Java**: Lenguaje principal utilizado para el desarrollo del backend.
- **JavaScript**: Lenguaje principal utilizado para el fontend.
- **HTML/CSS**: Para la estructura y estilo de las vistas generadas con Thymeleaf.
- **Maven**: Para la gestión de dependencias y construcción del proyecto.

## Frameworks y Librerías
- **Spring Boot**: Framework para el desarrollo de aplicaciones backend en Java.
- **Thymeleaf**: Motor de plantillas para generar vistas dinámicas en el frontend.
- **Grid.js**: Biblioteca para generar tablas con funciones de busqueda y ordenaciçon.
